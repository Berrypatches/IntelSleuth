OSINT Features

Search Engines

Scrapes data from DuckDuckGo, Bing, and other public search engines.

API Integration

Connects to IPinfo.io, Hunter.io, HaveIBeenPwned, and more with free-tier APIs.

WHOIS Lookup

Performs domain and IP WHOIS lookups to reveal registration details.

Smart Filtering

Filters out ads, duplicates, and irrelevant content from search results.

Categorization

Organizes findings into emails, phone numbers, social links, addresses, and more.

Webhook Export

Sends results to configurable webhook endpoints for n8n integration.

Usage Instructions

  1. Enter a query in the search box above (name, email, phone, username, domain, or IP)
  2. Optionally provide a webhook URL for n8n integration
  3. Click Search to begin the OSINT collection process
  4. Review the categorized results and executive summary
  5. Export results for further analysis or integration

Note: All searches are performed using public information and free API tiers. The quality of results may vary based on the availability of public data.
